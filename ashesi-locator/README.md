# Ashesi Resource Locator

## Overview
The Ashesi Resource Locator is a web application designed to help students and faculty at Ashesi University easily find and book campus resources. This application provides a user-friendly interface for navigating campus facilities, booking appointments, and accessing support services.

## Project Structure
```
ashesi-locator
├── src
│   ├── index.html        # Main HTML structure of the application
│   ├── css
│   │   └── styles.css    # CSS styles for the application
│   └── js
│       └── main.js       # JavaScript code for interactive features
└── README.md             # Documentation for the project
```

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/ashesi-locator.git
   cd ashesi-locator
   ```

2. **Open the Project**
   Open the `src/index.html` file in your web browser to view the application.

3. **Modify Styles**
   You can customize the styles by editing the `src/css/styles.css` file.

4. **Add JavaScript Functionality**
   For any interactive features, modify the `src/js/main.js` file.

## Usage Guidelines
- Navigate through the sidebar to access different sections of the application.
- Use the "Book a Resource" button to schedule appointments or find available resources.
- The application is designed to be responsive and should work well on both desktop and mobile devices.
